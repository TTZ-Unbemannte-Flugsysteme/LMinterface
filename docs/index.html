ROS2 + Edge LLM Project

Intelligent Drone Control via Natural Language

Empowering the Crazyflie nano quadcopter with spatial awareness, context grounding, and speech-to-text path execution using on-device Language Models.

Paper Abstract

This research presents a novel system architecture for controlling nano quadcopters (Crazyflie) using natural language speech inputs. By integrating Large Language Models (LLMs) running locally at the edge with the Robot Operating System (ROS2) and Software-in-the-Loop (SITL) simulations, the system parses natural voice commands into flight-critical task representations and routes them through A* collision-avoiding path planners.

We introduce Language-Guided Abstraction (LGA) and an Epistemic Knowledge Graph (EKG) to bind spoken command concepts to physical coordinate spaces. This grounding layer enables the quadcopter to dynamically interpret relative navigational references, conversational history, and spatial obstructions in warehouse environments.

"The empirical study demonstrates that small edge LLMs (7B parameters) require explicit semantic context structuring to perform reliable contextual reasoning. Without structured hints like CURRENT_LOCATION and PREVIOUS_LOCATION, the model defaults to pattern-matching from examples rather than true logical inference."

Core ROS2 Packages

The interface consists of modular ROS2 nodes that handle distinct layers of the translation, planning, and control lifecycle:

cocoa_dialogue
Hosts the web user interface. Captures microphone speech, transcribes it locally using Whisper ASR, and publishes commands.
cocoa_intent
Performs intent extraction and semantic tagging using structured instruction prompts sent to the edge-running LLM.
cocoa_ekg
Epistemic Knowledge Graph. Tracks environment objects, locations, states, and coordinates to ground the drone's world.
cocoa_lga
Language-Guided Abstraction. Filters and feeds only relevant spatial/state context to the LLM to fit within context limits.
cocoa_llm_planner
Processes high-level goal commands, dynamically structures target tasks, and manages drone action execution states.
cocoa_path_planner
Calculates collision-free waypoints around 3D structures like walls and shelves using an optimized A* search algorithm.
